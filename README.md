## Команды

Запуск dev-сервера Parcel (http://localhost:1234/): 

    npm run dev

Запуск сборки и сервера для раздачи статики (http://localhost:3000/):

    npm run start

Проверка кода компилятором tsc, линтерами ESLint и Stylelint:

    npm run lint
